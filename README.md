## 🎨 Visual Design & Features

- **Glassmorphic UI**: Beautiful semi-transparent frosted cards with harmonious, curated color palettes matching both light and dark modes.
- **Custom Interactive Background**: A dynamic particle-based Neural Network canvas background representing connection, intelligence, and modern AI styling.
- **Micro-Animations**: Elegant entrance animations and smooth hover transitions driven by `Framer Motion`.
- **Responsive Layout**: Designed mobile-first for optimized viewing across smartphones, tablets, and large screens.
- **Interactive Modals**: Seamless detailed views for featured projects and detailed academic background summaries.

## 🛠️ Local Development & Setup

To clone and run the portfolio locally on your system, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Jaasmitha2004/jaasmitha-peach.git
   cd jaasmitha-peach
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
